# ETL de Dados Meteorologicos

## 📖 Descrição
Projeto simples de ETL utilizando uma API publica, extraindo dados meteorologicos, tratando e carregando no banco PostgreSQL

## 🔄 Pipeline

MeteoSource API  
⬇️  
Extract  
⬇️  
Transform  
⬇️  
Load  
⬇️  
PostgreSQL

## 🛠️ Tecnologias Utilizadas

- 🐍 Python
- 📊 Pandas
- 🌐 Requests
- 🐘 PostgreSQL
- 🔗 SQLAlchemy
